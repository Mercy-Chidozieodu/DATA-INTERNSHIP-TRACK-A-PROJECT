# DATA-INTERNSHIP-TRACK-A-PROJECT
Data analytics projects using SQL and Powerbi, and is  focused on Sales analysis and Insights.
Internship Project A – Northwind Traders Retail and Supply chain


 **Project Overview****

This project focuses on analyzing the Northwind Traders dataset to uncover insights into sales performance, customer behavior, and retail supply chain operations.

The analysis was carried out using SQL and Power BI to transform raw data into meaningful business insights.


 **Tools & Technologies**

- SQL (Data extraction and transformation)
- Power BI (Data visualization and dashboard creation)
- Excel (Data cleaning and preparation)

---

📊 **Key Analysis Performed**

- Sales performance analysis
- Customer segmentation and behavior
- Product and category performance
- Regional sales trends
- Supply chain insights.

📈 **Dashboard Features**

- Interactive Power BI dashboard
- Dynamic filtering and drill-down capabilities
- KPI tracking (Sales, Orders, Customers)
- Clear visual representation of trends and patterns


💡 **Key Insights**

- Sales performance varies significantly across regions
- Certain products contribute more to overall revenue
- Customer purchasing patterns show repeat buying behavior
- Opportunities exist to optimize supply chain efficiency

📂 **Project Files**

- Power BI Dashboard (.pbix)
- SQL Scripts


🚀 Conclusion

This project demonstrates the use of data analytics tools to extract actionable insights that can support business decision-making and improve overall performance.
